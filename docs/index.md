# TORdex - Knowledge Intelligence Platform

> **TOR-Index [TORdex]**

TORdex is a **Knowledge Intelligence Platform** that combines concepts from:
- Palantir Gotham
- Palantir Foundry
- Maltego
- Recorded Future
- OpenCTI
- MISP
- Shodan
- Wayback Machine
- Sourcegraph
- Common Crawl
- Tor Browser Crawler

While remaining:
- ✅ **Local First**
- ✅ **Privacy First**
- ✅ **Event Driven**
- ✅ **Knowledge Centric**
- ✅ **Rust Native**

---

## Mission

Transform massive amounts of unstructured information into a living intelligence graph that can be **searched, analyzed, monitored, investigated, and automated**.

---

## Architecture Overview

TORdex is structured as a **20-layer stack**, where each layer adds value to the layer below it. Data enters at the bottom as raw information and leaves at the top as actionable intelligence.

### Layer Stack (0 → 19)

| Layer | Name | Purpose |
|-------|------|---------|
| 0 | **Sources** | Origin of all information |
| 1 | **Collection Fabric** | Gathers information from sources |
| 2 | **Collection Sessions** | Records what, when, and how data was collected |
| 3 | **Evidence Lake** | Preserves all collected data in original form |
| 4 | **Event Platform** | Records all internal system events |
| 5 | **Processing Fabric** | Transforms raw evidence into structured information |
| 6 | **Knowledge Core** | Central repository of normalized truth |
| 7 | **Knowledge DNA** | Unique identity via fingerprinting |
| 8 | **Correlation Engine** | Finds non-obvious connections |
| 9 | **Temporal Intelligence Graph** | Stores knowledge and relationships across time |
| 10 | **Repository Intelligence** | Deep analysis of software projects |
| 11 | **API Observatory** | Focused API discovery and monitoring |
| 12 | **Search Engine** | Gateway into the intelligence ecosystem |
| 13 | **Intelligence Engine** | Higher-level reasoning and insights |
| 14 | **Monitoring Engine** | Watches for changes and generates alerts |
| 15 | **Investigation Workspace** | Analyst workspace for research |
| 16 | **Intelligence Products** | Outputs: reports, briefings, analyses |
| 17 | **Agent Runtime** | Hosts automated analytical agents |
| 18 | **Decision Engine** | Supports critical decision-making |
| 19 | **Intelligence Console** | User-facing interface |
